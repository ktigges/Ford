Display Units

Logging

DEBUG writes verbose API/OAuth/poller details to the console and app log. INFO shows normal operational messages only. Per-module debug files (logs/debug_*.log) always capture DEBUG regardless of this setting.

Polling Intervals (seconds)

Moving: min 15s. All others: min 60s, max 3600s. Values outside these limits will be clamped.

When enabled, idle vehicles are still polled at the normal interval but records are only written to the database when the vehicle state changes or once every 60 minutes. Active states (ignition on, moving, charging) always write normally.

When enabled, the poller starts automatically when the Flask app boots.

EV Charger Network (NIL/NLR/NREL)

Connect to the NREL Alt Fuel Stations API to import and maintain a database of EV charging stations. This data is used for route planning and AI model training.

Get your free API key from NREL Developer Portal. Currently supports US chargers only.

Used when scope is "Single state only".

This endpoint ignores offset; this value is reserved for future API pagination support.
